# Спрайты игры

Вся графика игры — PNG-файлы в этой папке; атласов и спрайтов по умолчанию нет.
Файл с именем роли обязан существовать, иначе роль не отрисуется.

## Формат

- Имя файла задаёт роль (таблица ниже): `<роль>.png` — статичный спрайт,
  `<роль>.N.png` — анимация из N кадров, расположенных в изображении
  горизонтально слева направо (ширина кадра = ширина изображения / N).
- PNG с альфа-каналом.
- Базовая клетка — 32×32 px. Спрайты персонажей могут превышать клетку —
  якорь находится в середине её нижней грани. Роли `floor`, `path`, `wall`
  и `portal` масштабируются ровно в клетку.

## Роли

| Файл          | Назначение        |
|---------------|-------------------|
| `player.png`  | Игрок (Juggernaut) |
| `pudge.png`   | Враг: Pudge — медленный танк |
| `bloodseeker.png` | Враг: Bloodseeker — крадёт максимум HP |
| `riki.png`    | Враг: Riki — блинки и невидимость |
| `axe.png`     | Враг: Axe — двойной шаг и контратака |
| `skywrath.png` | Враг: Skywrath Mage — усыпляет, ходит по диагонали |
| `food.png`    | Предмет: еда (мотив Healing Salve) |
| `elixir.png`  | Предмет: эликсир (мотив Clarity) |
| `scroll.png`  | Предмет: свиток (мотив TP Scroll) |
| `sword.png`   | Предмет: оружие на полу (мотив Yasha) |
| `heart.png`   | UI: сердечко у HP-бара |
| `floor.png`   | Тайл травы (пол комнат) |
| `path.png`    | Тайл тропы (коридоры) |
| `wall.png`    | Тайл чащи (стены, туман, пустота) |
| `decor.png`   | Кустики, изредка рисуются на траве |
| `tree.png`    | Деревья, изредка рисуются в чаще |
| `portal.png`  | Портал — выход на следующий уровень |

Для тайловых ролей (`floor`, `path`, `wall`, `portal`) кадры мультикадрового PNG —
пространственные варианты: выбираются по хешу координат клетки, а не по времени.
