# Свои спрайты (Dota-style)

Положи сюда PNG — и он автоматически заменит спрайт в игре. Остальное не трогается,
можно рисовать по одному персонажу за раз и сразу видеть результат.

## Правила

- **Размер**: рисуй **32×32 px** (это ровно одна клетка). Персонажи могут быть выше
  (например 32×40) — «лишняя» высота торчит над клеткой, ноги стоят на полу.
- **Формат**: PNG с прозрачным фоном (альфа-канал).
- **Имя файла = роль** (см. список ниже). Пример: `player.png` заменит игрока.
- **Анимация**: сложи кадры в один PNG по горизонтали (слева направо) и укажи их
  число в имени через точку: `player.4.png` = 4 кадра по 32px = картинка 128×32.
  Без числа (`player.png`) — статичный спрайт в один кадр.

## Роли

| Файл          | Кто это           |
|---------------|-------------------|
| `player.png`  | Игрок (Juggernaut) |
| `zombie.png`  | Dire Ghoul        |
| `lifestealer.png` | Lifestealer   |
| `spectre.png` | Spectre           |
| `ogre.png`    | Ogre Magi         |
| `medusa.png`  | Medusa            |
| `food.png`    | Еда (Salve)       |
| `elixir.png`  | Эликсир (Bottle)  |
| `scroll.png`  | Свиток (Tome)     |
| `sword.png`   | Оружие на полу    |
| `coin.png`    | Монеты/сокровища  |
| `floor.png`   | Пол (кроет всю карту) |
| `wall.png`    | Стены             |
| `ladder.png`  | Лестница вниз (выход) |

## С чего начать

1. Открой https://piskelapp.com (бесплатно, в браузере).
2. Задай размер холста 32×32.
3. Нарисуй, например, Джаггернаута: тёмно-синий плащ, белая маска, меч.
4. Экспортируй как PNG, назови `player.png`, положи в эту папку.
5. Запусти игру (`python game/main.py`) — увидишь свой спрайт.
6. Довёл до ума — коммить и пуш, CI задеплоит на сервер.
