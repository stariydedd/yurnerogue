# YurneROGUE

Pygame-переосмысление классического Rogue в стилистике Dota 2 — движок
подземелья с графическим интерфейсом, собирается в браузерную WASM-версию через
[pygbag](https://github.com/pygame-web/pygbag).

Игра доступна в браузере: http://194.58.46.226 — глобальный лидерборд общий
для всех игроков.

## Лор

Тёмный Карнавал прошёл через джунгли и не ушёл целиком: его твари остались и
исказили лес. Юрнеро прорубает поляны в чаще и спускается всё глубже — двадцать
один круг вниз, к сердцу Карнавала. С каждым кругом лес темнее, твари сильнее,
а зелье и сталь встречаются реже. Каждый удар может стать последним — вопрос
только, чьим.

## Архитектура

- **`game/domain/`** — правила игры (сессия, уровни, бой, враги, предметы),
  не зависит от способа отрисовки.
- **`game/presentation/`** — рендер на `pygame-ce`: тайлы карты, статус-панель, меню,
  диалоги. Конечный автомат состояний в `view.py` обрабатывает одно событие за кадр.
- **`game/datalayer/`** — локальные сохранения и клиент глобального лидерборда.
- **`game/main.py`** — точка входа с `asyncio`-циклом (требование pygbag).

## Установка и запуск

```
python -m venv .venv
.venv\Scripts\Activate.ps1      # Windows
pip install -r requirements.txt
python game/main.py
```

## Управление

| Клавиша   | Действие          |
|-----------|-------------------|
| `W A S D` / стрелки | Движение |
| `H`       | Оружие            |
| `J`       | Еда               |
| `K`       | Эликсир           |
| `E`       | Свиток            |
| `F1`      | Справка           |
| `Q`       | Возврат в меню    |

## Противники

| Враг | Поведение |
|------|-----------|
| **Pudge** | Медлительный и живучий. Ходит случайно по 4 направлениям. |
| **Bloodseeker** | При ударе крадёт максимум здоровья (1/10 за удар). Первая атака игрока по нему отражается. Ходит по 8 направлениям. |
| **Skeleton Ghost** | Телепортируется в пределах комнаты, большую часть времени невидим (появляется с шансом 20% за ход, при погоне виден всегда). |
| **Axe** | Ходит на 2 клетки за ход. После атаки ход отдыхает, затем контратакует. Его удар нельзя увернуться. |
| **Skywrath Mage** | Ходит только по диагонали и бьёт по диагонали. Удар с шансом 15% усыпляет на ход. |

С каждым уровнем характеристики врагов растут, а полезных предметов становится меньше.

## Предметы

| Предмет | Эффект |
|---------|--------|
| Еда (красная фляжка) | Восстанавливает здоровье. |
| Эликсир (синяя фляжка) | Временный бафф к силе, ловкости или максимуму HP на 20 ходов. |
| Свиток (жёлтая фляжка) | Постоянный бафф к одной из характеристик. |
| Оружие (меч) | Экипируется через `H`; прежнее оружие падает рядом на землю. |
| Сокровища | Выпадают из убитых врагов; определяют место в лидерборде. |

Предметы подбираются автоматически при наступании; рюкзак вмещает до 9 предметов
каждого типа.

## Выход

Лестница на поляне — спуск на следующий уровень. Всего 21 уровень; спуск после
21-го — победа. Результат забега (смерть или победа) отправляется в глобальный
лидерборд.
